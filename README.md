# RL_Q-Learning_Agent
This project implements a Q-learning agent that learns to navigate a 10×10 grid environment. The agent's objective is to reach a randomly placed goal while avoiding an enemy. States are represented by the relative positions of the goal and the enemy with respect to the agent. During training, the agent uses an epsilon-greedy exploration strategy with adaptive exploration and learns through Q-table updates based on rewards that encourage reaching the goal, avoiding the enemy, and moving efficiently. Training performance is tracked by measuring the success rate over multiple episodes, and the environment can be visualized in real time using OpenCV. Project designed and created during smart materials and methods lessons at university.
